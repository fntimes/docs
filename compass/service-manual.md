# COMPASS 서비스 매뉴얼

**기업 경영성과 비교 분석 플랫폼**

---

## 1. 서비스 소개

COMPASS는 비교 가능한 기업들을 하나의 리그로 묶어, 주요 경영지표를 **순위표**와 **시계열 그래프**로 제공하는 데이터 분석 플랫폼입니다. 공시 자료를 일일이 수집·정리할 필요 없이, 기업 간 실적을 즉시 비교하실 수 있습니다.

## 3. 메뉴 구조

```
COMPASS
├── 경영성과
│   ├── 4대 금융지주       ← 리그테이블 / 트렌드
│   ├── 4대 시중은행       ← 리그테이블 / 트렌드
│   ├── 증권사             ← 리그테이블(기업별·지표별) / 트렌드

├── 시장분석 (개발중)
│   ├── DCM (회사채)
│   └── 퇴직연금
│
├── 피드백                  ← 의견 등록
└── 서비스 매뉴얼           ← 현재 페이지
```

## 4. 핵심 기능 상세

COMPASS는 **리그테이블**, **트렌드**, **AI 인사이트** 세 가지 핵심 기능을 제공합니다.

### 4-1. 리그테이블 — 지표별 순위 비교

리그테이블은 선택한 지표에 대해 기업들의 순위를 한눈에 보여줍니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 지표 테이블 (좌측) | 부문별로 그룹화된 지표 목록. 각 기업의 수치와 순위 표시 |
| 상세 차트 패널 (우측) | 지표를 클릭하면 나타나는 비교 차트와 업계 평균 대비 차트 |
| 1위 하이라이트 | 금색 배경으로 1위 기업 강조 |
| 업계 평균 | 비교 대상 기업들의 평균값 |
| 평균 증감 | 전년 동기 대비 업계 평균의 증감 (청록/적색 표시) |
| 격차 | 1위와 최하위 기업 간의 격차 |

**지표 클릭 시 사이드 패널에서 확인 가능한 차트:**
- **비교 차트**: 현재 분기 vs 비교 기간(전분기/전년 동기) 막대 그래프
- **평균 대비 편차 차트**: 업계 평균 대비 각 기업의 위치

**사이드 패널 조작법:**
- 기간 유형 전환: `분기` / `누적` (해당 지표에 누적 데이터가 있는 경우)
- 비교 기준 전환: `전분기` / `전년동기`
- 차트별 PNG 다운로드 가능
- `←` 버튼으로 패널 닫기

---

### 4-2. 트렌드 — 시계열 추이 분석

트렌드는 2021년부터 현재까지의 지표 변화를 시계열 그래프로 보여줍니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 지표 선택 (좌측 사이드바) | 부문별로 그룹화된 지표 목록. 클릭하여 지표 전환 |
| 데이터 테이블 (상단) | 선택한 지표의 분기별 수치를 테이블로 표시 |
| 스몰 멀티플 차트 (좌측 하단) | 기업별 개별 추이 그래프 (2x2 격자 형태) |
| 순위 변동 차트 (우측 하단) | 기업 간 순위 변화를 범프 차트로 표시 |

**기간 설정 버튼:**

| 버튼 | 설명 |
|------|------|
| 1Y | 최근 1년 (4개 분기) |
| 2Y | 최근 2년 (8개 분기) — 기본 선택 |
| 3Y | 최근 3년 (12개 분기) |
| 5Y | 최근 5년 (20개 분기) |
| 연간 | 연도별 연간 합산 데이터 |

**추가 조작:**
- `분기` / `누적` 전환: 지표의 기간 유형 선택
- `Excel 다운로드`: 현재 선택한 지표의 트렌드 데이터를 엑셀로 내려받기
- 차트별 PNG 다운로드 가능

---

### 4-3. AI 인사이트 — 데이터 기반 자동 분석

AI 인사이트는 화면에 표시된 데이터를 AI가 분석하여 핵심 포인트를 정리합니다.

**사용 방법:**
1. 리그테이블 또는 트렌드 화면 상단의 `AI 분석` 버튼 클릭
2. AI 분석 모달이 열리며 자동으로 분석이 시작됩니다

**모달 구성:**

| 영역 | 설명 |
|------|------|
| 인사이트 패널 (좌측) | AI가 생성한 분석 결과 |
| 채팅 패널 (우측) | 추가 질문을 위한 대화 인터페이스 |

**인사이트 패널 제공 항목:**
- **종합 요약**: 전체적인 순위와 핵심 수치 정리
- **주요 기업 프로필**: 각 기업의 특징과 강약점
- **교차 분석 인사이트**: 부문 간 연관 분석
- **기사 앵글 제안**: 기사 작성에 활용할 수 있는 관점 제안

각 항목에는 `복사` 버튼이 있어, 필요한 분석 결과를 바로 복사하여 활용할 수 있습니다.

**채팅 패널 사용법:**
- 분석이 완료되면 하단 입력창이 활성화됩니다
- 미리 준비된 추천 질문을 클릭하거나 직접 질문을 입력합니다
- 예: "우리금융이 3위로 올라가려면 어느 정도 실적이 필요해?"

**AI 모델 선택:**
- 모달 상단에서 AI 모델을 전환할 수 있습니다
- `재생성` 버튼으로 분석을 다시 생성할 수 있습니다

## 5. 리그별 상세 가이드

### 5-1. 4대 금융지주

**비교 대상**: KB금융, 신한지주, 하나금융지주, 우리금융지주

**평가 부문 및 주요 지표:**

| 부문 | 주요 지표 |
|------|----------|
| 성장성 | 총자산, 총자산 증감률, 계열사순이익, 비은행순이익, 비은행비중 |
| 수익성 | 영업이익, 이자수익, 비이자수익, 당기순이익, NIM, ROA, ROE |
| 건전성 | 부실채권(NPL), NPL비율, NPL충당비율, 대손충당금, 신용비용률(CCR), 연체율, 위험가중자산(RWA), RORWA |
| 자본적정성 | BIS비율, CET1비율, Tier1비율 |
| 효율성 | 영업이익경비율(CIR), 노동비당수익 |

**리그테이블 특징:**
- 4개 기업 + 4사 평균 + 평균 증감 + 격차 칼럼
- 지표 클릭 시 우측에 비교 차트와 평균 대비 편차 차트 표시

**트렌드 특징:**
- 4개 기업의 분기별 추이를 스몰 멀티플(2x2)과 순위 변동 차트로 표시
- 2021년~현재 데이터

### 5-3. 26개 증권사

**비교 대상**: 상장 20개사 + 비상장 6개사 (총 26개 증권사)

**평가 부문 및 주요 지표:**

| 부문 | 주요 지표 |
|------|----------|
| 성장성 | 총자산, 순영업수익, 시장점유율(위탁매매) |
| 수익성 | 영업이익, 당기순이익, ROE, ROA, 수수료수익비중 |
| 건전성 | 순자본비율(NCR), 레버리지비율, 고정이하자산비율 |
| 효율성 | CIR, 1인당 순영업수익 |
| 주주환원 | EPS, BPS, PER, PBR, TSR, 주주환원율, 배당성향 (상장사 한정) |

**증권사만의 특별 기능:**

#### 두 가지 보기 모드

1. **기업별 보기** (기본)
   - 26개 증권사를 행(row)으로, 전체 지표를 열(column)으로 배치
   - 정렬: 지표 헤더를 클릭하면 해당 지표 기준으로 오름차순/내림차순 정렬
   - 기업 선택: `기업 선택` 버튼으로 표시할 증권사를 선택 (최대 10개)
     - 빠른 선택: `TOP 10` / `금융지주 계열` 버튼
   - 1위 기업은 금색 배경으로 강조

2. **지표별 보기**
   - 좌측에 지표 목록, 우측에 상위 10개 증권사의 순위를 표시
   - 지표 클릭 시 사이드 패널에 상위 10개사 비교 막대 차트 표시

#### 회계 기준 전환

- `연결` / `별도` 버튼으로 연결재무제표/별도재무제표 기준 전환

#### 기간 유형 전환

- `분기(Q)` / `누적(YTD)` 버튼으로 분기 실적/누적 실적 전환

## 6. 데이터 다운로드

### Excel 다운로드

각 화면에서 현재 표시된 데이터를 엑셀 파일로 내려받을 수 있습니다.

| 화면 | 다운로드 내용 |
|------|-------------|
| 리그테이블 | 전 지표의 수치, 전분기/전년동기 대비 증감 |
| 트렌드 | 선택한 지표의 분기별 시계열 데이터 |
| 증권사 리그테이블 | 26개 증권사 전체 지표 일괄 다운로드 |

**다운로드 방법:**
1. 화면 우측 상단의 `Excel 다운로드` 버튼 클릭
2. 엑셀 파일이 자동으로 다운로드됩니다

### 차트 PNG 다운로드

사이드 패널이나 트렌드 화면의 차트에서 개별 차트를 PNG 이미지로 저장할 수 있습니다.
- 차트 우측 상단의 다운로드 아이콘 클릭

## 7. AI 토큰 안내

AI 분석 기능은 외부 유료 AI 서비스를 활용하므로 월간 사용량 제한이 있습니다.

| 항목 | 내용 |
|------|------|
| 월간 한도 | 사용자당 100,000 토큰 |
| 리셋 시점 | 매월 1일 자동 리셋 |
| 사용량 확인 | AI 모달 내 채팅 패널 하단에 잔여량 표시 |

**참고:**
- 일반적인 사용 패턴(하루 5~10회 분석 요청)으로 한 달 동안 충분히 사용 가능합니다
- 한도 초과 시 다음 달 1일까지 AI 기능 사용이 제한됩니다
- 추가 사용량이 필요한 경우 관리자에게 요청하시면 추가 부여가 가능합니다

## 8. 접속 방법

COMPASS는 현재 외부 공개 서비스가 아닌 **사내 임직원 전용 서비스**입니다. 서비스 안정화와 내부 피드백 반영을 위해 사내 네트워크와 VPN을 통한 접속만 허용하고 있습니다.

### 사내 네트워크

사내 Wi-Fi 또는 유선 네트워크에서는 VPN 없이 바로 접속 가능합니다.

### 외부 접속 (VPN)

외부 네트워크에서는 VPN 연결이 필요합니다.

**VPN 설정 (최초 1회)**

| 단계 | 조작 |
|------|------|
| 1 | Outline 클라이언트 설치 — [Windows](https://s3.amazonaws.com/outline-releases/client/windows/stable/Outline-Client.exe) / [macOS](https://apps.apple.com/app/outline-app/id1356178125) |
| 2 | Outline 실행 → 관리자에게 전달받은 액세스 키(`ss://...`) 입력 |
| 3 | `연결` 버튼 클릭 → 화면이 녹색으로 바뀌면 완료 |

> **주의**: VPN 사용량이 한정되어 있습니다. COMPASS 사용 완료 후 반드시 `연결 해제`를 눌러주세요.

**VPN 문제 해결**

| 증상 | 해결 방법 |
|-----|----------|
| 연결 버튼을 눌러도 연결되지 않음 | V3, 알약 등 백신 프로그램을 잠시 종료 후 재시도 |
| 액세스 키가 유효하지 않다고 표시됨 | 관리자에게 새 액세스 키 요청 |
| COMPASS 페이지가 열리지 않음 | VPN 연결 상태 확인 (녹색 표시 여부) |

### 로그인

등록된 이메일 주소와 비밀번호로 로그인합니다. 초기 비밀번호는 관리자가 별도 안내합니다.

### 비밀번호 변경

**최초 로그인 시 비밀번호 변경이 필수입니다.** 변경하지 않으면 서비스를 이용할 수 없습니다.

- 최초 로그인 시 변경 화면이 자동 표시됩니다
- 새 비밀번호: 8자 이상, 영문과 숫자 포함
- 비밀번호 분실 시 관리자에게 문의

### 앱으로 설치 (선택)

별도 앱 설치 없이 웹 브라우저에서 이용 가능하지만, 바탕화면 바로가기를 추가할 수 있습니다.

1. COMPASS 사이트 접속
2. 화면 하단의 `앱으로 설치` 버튼 클릭
3. 바탕화면 아이콘으로 바로 접속 가능

## 9. 자주 묻는 질문 (FAQ)

**Q. 데이터는 얼마나 자주 업데이트되나요?**
> 각 기업의 분기 실적 공시에 맞춰 업데이트됩니다. 데이터 기준 시점은 각 화면 상단에 표시됩니다.

**Q. 리그테이블에서 금색 배경은 무엇인가요?**
> 해당 지표에서 1위인 기업을 나타냅니다.

**Q. 증권사 리그테이블에서 특정 증권사만 보고 싶습니다.**
> 상단 `기업 선택` 버튼을 클릭하여 원하는 증권사를 선택(최대 10개)하거나, `TOP 10` / `금융지주 계열` 버튼으로 빠르게 필터링할 수 있습니다.

**Q. 트렌드 화면에서 연간 데이터를 보고 싶습니다.**
> 기간 설정 버튼 중 `연간`을 클릭하면 분기 데이터가 연도별로 합산되어 표시됩니다.

**Q. AI 토큰이 모두 소진되면 어떻게 되나요?**
> AI 분석 기능만 제한되며, 리그테이블과 트렌드 등 데이터 조회 기능은 정상적으로 이용 가능합니다. 다음 달 1일에 자동 리셋되며, 긴급 시 관리자에게 추가 부여를 요청할 수 있습니다.
