# Compass 프로젝트 개발 로드맵

Compass 시스템 1차 오픈을 위한 개발 로드맵.
4대 금융지주 경영성과 데이터 수집 및 비교 분석 대시보드 구축.

---

## 1차 오픈 범위

| 항목 | 내용 |
|-----|------|
| 대상 기업 | KB금융, 신한금융, 하나금융, 우리금융 |
| 수집 지표 | 44개 경영성과 지표 |
| 주요 기능 | 지표 조회, 금융지주 비교, 데이터 다운로드 |

---

## 전체 일정

| 단계 | 기간 | 핵심 목표 |
|:---:|:----:|----------|
| 1 | 1-2주 | 데이터 기반 구축 |
| 2 | 3-7주 | 데이터 수집 |
| 3 | 8주 | 관리자 대시보드 구현 |
| 4 | 9-10주 | 사용자 대시보드 구현 |
| 5 | 11-12주 | 테스트 및 오픈 |

---

## 주차별 상세 일정

### 1단계: 데이터 기반 구축 (1-2주)

| 주차 | 작업 | 세부 내용 |
|:---:|------|----------|
| 1주 | 데이터베이스 설계 | 금융지주, 지표, 수집 이력 테이블 구성 |
| 2주 | 데이터 소스 검증 | 수집 가능 여부 및 추출 방식 확정 |
| 2주 | 기초 데이터 입력 | 기업 개황, 업종, 테마그룹, 지표 정의 등록 |

---

### 2단계: 데이터 수집 (3-7주)

| 주차 | 작업 | 세부 내용 |
|:---:|------|----------|
| 3-4주 | 기본 지표 수집 시스템 개발 | 4대 금융지주 기본 지표 추출 |
| 5주 | 파생 지표 산출 시스템 개발 | 증감률, 비중 등 계산이 필요한 지표 자동 산출 |
| 6주 | 외부 데이터 연동 시스템 개발 | 필수 지표 산출을 위한 주가 등 외부 정보 연결 |
| 7주 | 수집 결과 검증 및 보완 | 4사 동일 기준 비교, 이상치 점검, 예외 처리 |

---

### 3단계: 관리자 대시보드 구현 (8주)

| 주차 | 작업 | 세부 내용 |
|:---:|------|----------|
| 8주 | 수집 현황 모니터링 화면 | 수집 상태, 오류 관리, 수동 재수집 |
| 8주 | 데이터 관리 기능 | 수동 데이터 추가/수정/삭제 |
| 8주 | 스케줄러 설정 | 분기별 IR 자료, 일별 주가 자동 수집 |

---

### 4단계: 사용자 대시보드 구현 (9-10주)

| 주차 | 작업 | 세부 내용 |
|:---:|------|----------|
| 9주 | 금융지주별 상세 화면 구현 | 지표 카드, 분기별 추이 차트 |
| 9주 | 시계열 차트 구현 | 지표별 트렌드 시각화 |
| 10주 | 비교 차트 구현 | 4사 비교 테이블, 레이더 차트 |
| 10주 | 데이터 다운로드 기능 | Excel, CSV 내보내기 |

---

### 5단계: 테스트 및 오픈 (11-12주)

| 주차 | 작업 | 세부 내용 |
|:---:|------|----------|
| 11주 | 클라우드 인프라 구성 | AWS 기반 서버 및 데이터베이스 구축 |
| 11주 | 통합 테스트 및 성능 개선 | 전체 기능 점검, 화면 로딩 최적화 |
| 12주 | 운영 환경 배포 | 도메인 연결, 보안 설정, 모니터링 |
| 12주 | 서비스 오픈 | 최종 점검 후 오픈 |
