## 1. 서비스 소개

### FnBook이란?

FnBook은 기업 연락처 및 기념일을 효율적으로 관리하기 위한 통합 관리 시스템입니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| 연락처 관리 | 기업별 담당자 연락처 등록, 수정, 검색 |
| 기업 정보 관리 | 상장사/비상장사 기업 데이터베이스 관리 |
| 기념일 관리 | 창립기념일, 상장기념일 등 주요 일정 캘린더 관리 |
| 엑셀 연동 | 연락처 대량 임포트 및 다운로드 |

---

## 2. 시작하기

### 2.1 로그인

1. 웹 브라우저에서 FnBook 접속
2. 이메일 주소와 비밀번호 입력
3. **로그인** 버튼 클릭

> **참고**: 3분 내 10회 이상 로그인 실패 시 일시적으로 접근이 제한됩니다.

### 2.2 화면 구성

#### 데스크톱

| 영역 | 설명 |
|------|------|
| 좌측 사이드바 | 메인 메뉴 네비게이션 및 사용자 정보 |
| 상단 영역 | 페이지 제목 및 주요 액션 버튼 |
| 메인 영역 | 콘텐츠 표시 |

#### 모바일

| 영역 | 설명 |
|------|------|
| 상단 헤더 | 현재 페이지 정보 |
| 하단 탭바 | 주요 메뉴 빠른 접근 |
| 메인 영역 | 카드 형식 콘텐츠 |

## 3. 메뉴별 기능 안내

### 3.0 비밀번호 변경

로그인한 사용자 본인의 비밀번호를 변경할 수 있습니다.

#### 접근 방법

| 플랫폼 | 방법 |
|--------|------|
| 데스크톱 | 좌측 하단 프로필 영역의 설정 아이콘(톱니바퀴) 클릭 → **비밀번호 변경** 선택 |
| 모바일 | 하단 **더보기** 탭 → 계정 섹션에서 **비밀번호 변경** 선택 |

#### 비밀번호 변경 방법

1. **비밀번호 변경** 클릭
2. **현재 비밀번호** 입력 (본인 확인용)
3. **새 비밀번호** 입력 (최소 6자 이상)
4. **새 비밀번호 확인** 입력
5. **비밀번호 변경** 버튼 클릭

> **참고**: 비밀번호 변경 시 현재 비밀번호가 일치해야 합니다.

### 3.1 홈 (대시보드)

홈 화면에서는 시스템 전체 현황을 한눈에 확인할 수 있습니다.

#### 통계 카드

| 항목 | 설명 |
|------|------|
| 전체 기업 | 등록된 총 기업 수 |
| 전체 연락처 | 등록된 총 연락처 수 |
| 이번 달 기념일 | 이번 달에 있는 기념일 수 |
| 이번 주 기념일 | 이번 주에 있는 기념일 수 |

#### 부서별/업종별 연락처

| 섹션 | 설명 |
|------|------|
| 부서별 연락처 | 담당부서별 연락처 수 확인 및 빠른 필터링 |
| 업종별 연락처 | 업종(카테고리)별 연락처 수 확인 및 빠른 필터링 |

#### 다가오는 기념일

| 표시 | 설명 |
|------|------|
| 기본 | 우측 패널에서 예정된 기념일 타임라인 확인, 월별 그룹화 |
| 오늘 기념일 | 파란색 배경으로 강조 |
| 주요 주년 (마일스톤) | 1주년, 5주년, 10주년 등 5년 단위 기념일은 골드색 뱃지로 강조 |

---

### 3.2 연락처

기업 담당자의 연락처를 관리하는 메뉴입니다.

#### 연락처 목록 조회

- 테이블 형식으로 전체 연락처 확인
- 이름, 직위, 소속기업, 담당부서, 연락처 정보 표시

#### 검색 및 필터

| 필터 | 설명 |
|------|------|
| 담당부서 | 특정 부서가 담당하는 연락처만 표시 |
| 업종 | 특정 업종(카테고리)의 연락처만 표시 |
| 임원만 | 임원으로 표시된 연락처만 표시 |
| 검색 | 이름, 회사명, 연락처로 검색 |

#### 연락처 등록

1. **+ 연락처 등록** 버튼 클릭
2. 필수 정보 입력:
   - **이름** (필수)
   - **소속기업** (필수): 드롭다운에서 선택
   - **담당부서** (필수): 해당 연락처를 담당할 부서 선택
3. 선택 정보 입력:
   - 직위, 부서명, 직무
   - 회사전화, 팩스, 휴대전화
   - 이메일, 보조 이메일
   - 업종 그룹, 임원 여부, 메모
4. **저장** 버튼 클릭

#### 연락처 수정/삭제

- 목록에서 해당 연락처의 **수정** 또는 **삭제** 버튼 클릭
- 삭제된 연락처는 복구할 수 없으니 주의

#### 연락처 상태

| 상태 | 설명 |
|------|------|
| 활성 (Active) | 현재 재직 중인 담당자 |
| 퇴직 (Resigned) | 퇴직한 담당자 |
| 전직 (Transferred) | 다른 회사로 이직한 담당자 |

#### 엑셀 다운로드

1. **다운로드** 버튼 클릭
2. 부서별 필터 선택 (선택사항)
3. **다운로드** 버튼으로 엑셀 파일 저장

> **참고**: 편집국 소속 직원은 부장급 이상만 다운로드가 가능합니다.

#### 엑셀 임포트 (대량 등록)

1. **임포트** 버튼 클릭
2. 양식 다운로드 후 데이터 입력
3. 엑셀 파일 업로드
4. 미리보기에서 데이터 확인 (유효/오류 표시)
5. **임포트 실행** 버튼 클릭

**엑셀 양식 필드**:

| 필드명 | 필수 | 설명 |
|--------|------|------|
| 이름 | O | 담당자 이름 |
| 소속기업 | O | 기업명 (시스템에 등록된 기업명과 일치해야 함) |
| 직위 | - | 예: 대표이사, 부장, 과장 |
| 그룹 | - | 업종 그룹명 |
| 부서 | - | 담당자의 소속 부서명 |
| 직무 | - | 담당 업무 |
| 회사전화 | - | 회사 대표번호 |
| 팩스 | - | 팩스 번호 |
| 휴대전화 | - | 개인 휴대전화 |
| 이메일 | - | 주 이메일 |
| 보조이메일 | - | 보조 이메일 |
| 임원 | - | Y 또는 N |
| 메모 | - | 기타 메모 |

#### 수정 이력 확인

연락처 상세 화면에서 **수정 이력** 버튼을 클릭하면 최근 10건의 변경 내역을 확인할 수 있습니다.

### 3.3 기업

기업 정보를 관리하는 메뉴입니다.

#### 기업 목록 조회

- 기업명, 업종, 대표번호, 대표자, 설립일 등 확인
- 상장유형별 필터링 가능

#### 상장유형

| 유형 | 설명 |
|------|------|
| 유가증권 | 유가증권시장(KOSPI) 상장 |
| 코스닥 | KOSDAQ 상장 |
| 코넥스 | KONEX 상장 |
| 기타 | 비상장 또는 기타 |

#### 기업 등록

1. **+ 기업 등록** 버튼 클릭
2. 기업 정보 입력:
   - **기업명** (필수)
   - 영문명, 상장유형, 종목코드
   - 대표자명, 설립일
   - 주소, 대표번호, 팩스
   - 웹사이트 URL, IR 페이지 URL
3. **저장** 버튼 클릭

#### 기업 상세 정보

기업을 클릭하면 상세 페이지에서 다음을 확인할 수 있습니다:

- 기업 기본 정보
- 해당 기업 소속 연락처 목록
- 창립기념일 정보

#### 창립기념일 자동 생성

- 설립일을 입력하면 창립기념일이 자동으로 생성됩니다
- 상장사는 기본적으로 캘린더에 표시됩니다
- 기업 상세 페이지에서 표시 여부를 변경할 수 있습니다

---

### 3.4 기념일

기업의 주요 기념일을 캘린더로 관리하는 메뉴입니다.

#### 캘린더 보기

- 월별 캘린더 형식으로 기념일 표시
- 이벤트 클릭으로 상세 정보 확인
- 좌우 화살표로 월 이동
- **주요 주년 (마일스톤)**: 1주년, 5주년, 10주년 등은 골드색 이벤트 바로 강조 표시

#### 기념일 모니터링

기념일 목록 페이지에서 다가오는 기념일을 기간별로 확인할 수 있습니다:

| 기간 | 표시 형태 | 강조 색상 |
|------|----------|----------|
| 7일 이내 | 카드 형태로 상세 정보 표시 | 빨간색 |
| 30일 이내 | 테이블/리스트 형태 | 주황색 |
| 이번 분기 | 칩 형태로 간략 표시 | 파란색 |

#### 기념일 유형

| 유형 | 설명 |
|------|------|
| 창립 | 회사 창립기념일 |
| 상장 | 주식시장 상장 기념일 |
| 인수합병 | M&A 완료 기념일 |
| 피인수 | 피인수 기념일 |
| 사옥이전 | 본사 이전 기념일 |
| 사명변경 | 회사명 변경 기념일 |
| CEO취임 | 대표이사 취임 기념일 |
| IR행사 | IR 행사 일정 |
| 주주총회 | 정기/임시 주주총회 |
| 사용자정의 | 기타 사용자 지정 기념일 |

#### 기념일 등록

1. **+ 기념일 등록** 버튼 클릭
2. 정보 입력:
   - **기업** 선택 (필수)
   - **기념일명** 입력 (필수)
   - **기념일 유형** 선택
   - **기준일** 선택 (필수)
   - 반복 설정 (매년/매월)
   - 캘린더 표시 여부
   - 알림 설정
3. **저장** 버튼 클릭

#### 반복 설정

| 설정 | 설명 |
|------|------|
| 매년 반복 | 매년 같은 날짜에 반복 (기본값) |
| 매월 반복 | 매월 같은 날짜에 반복 |
| 반복 없음 | 일회성 일정 |

#### 알림 설정

| 항목 | 설명 |
|------|------|
| 알림 활성화 | 알림 수신 여부 |
| 알림 기준 | 기념일 며칠 전에 알림 (1~90일) |

#### 주요 주년 (마일스톤)

시스템은 자동으로 주년을 계산하여 표시합니다. 다음 주년은 골드색 뱃지로 강조 표시됩니다:

- 1주년, 5주년, 10주년, 15주년, 20주년, 25주년, 30주년, 35주년, 40주년, 45주년, 50주년 등

### 3.5 관리 메뉴

> **접근 권한**: 관리자(Admin) 또는 부서장(Department Head)만 접근 가능

#### 사용자 관리

시스템 사용자를 관리합니다.

**사용자 목록**
- 이름, 이메일, 역할, 소속부서 확인
- 역할별, 부서별 필터링

**사용자 등록**
1. **+ 사용자 등록** 버튼 클릭
2. 정보 입력:
   - **이름** (필수)
   - **이메일** (필수, 로그인 ID로 사용)
   - **비밀번호** (필수)
   - **역할** 선택
   - 소속부서, 직위, 직급
3. **저장** 버튼 클릭

> **참고**: 사용자 생성/수정/삭제는 관리자(Admin)만 가능합니다.

#### 부서 관리

> **접근 권한**: 관리자(Admin)만 접근 가능

조직의 부서 구조를 관리합니다.

**부서 구조**

| 구분 | 설명 |
|------|------|
| 본부(Division) | 상위 조직 단위 |
| 팀(Team) | 본부 하위의 팀 단위 |

**부서 등록**
1. **+ 부서 등록** 버튼 클릭
2. 정보 입력:
   - **부서명** (필수)
   - **부서코드** (필수, 영문)
   - **구분** (본부/팀)
   - 상위부서 (팀인 경우)
3. **저장** 버튼 클릭

## 4. 권한 및 역할

### 4.1 역할 종류

| 역할 | 영문명 | 설명 |
|------|--------|------|
| 관리자 | Admin | 모든 기능 접근 및 시스템 관리 |
| 부서장 | Department Head | 부서 관리 및 사용자 메뉴 접근 |
| 팀장 | Team Head | 일반 조회/입력 기능 |
| 기자 | Reporter | 일반 조회/입력 기능 |
| 열람자 | Viewer | 읽기 전용 |

### 4.2 역할별 권한 상세

#### 관리자 (Admin)

- 모든 메뉴 접근
- 연락처 다운로드 제한 없음
- 사용자 생성/수정/삭제
- 부서 관리

#### 부서장 (Department Head)

- 일반 기능 모두 사용
- 사용자 메뉴 접근 (열람)
- 연락처 다운로드 (편집국 소속 시 직급 조건 적용)

#### 팀장/기자/열람자

- 연락처, 기업, 기념일 조회 및 등록
- 연락처 다운로드 (편집국 소속 시 직급 조건 적용)

### 4.3 연락처 다운로드 권한

연락처 엑셀 다운로드는 다음 조건으로 제한됩니다:

| 소속 | 조건 | 다운로드 가능 여부 |
|------|------|-------------------|
| 관리자 | - | O |
| 편집국 소속 | 부장급 이상 | O |
| 편집국 소속 | 부장급 미만 | X |
| 기타 부서 | - | O |

**편집국 부서**: 금융부, 증권부, 산업부, 유통부, 부동산부

**부장급 이상 직급**: 부장, 국장, 이사, 전무, 사장

## 5. 자주 묻는 질문

### Q1. 비밀번호를 변경하고 싶어요.

**본인이 직접 변경하는 방법:**
1. 좌측 하단 프로필 영역의 설정 아이콘(톱니바퀴) 클릭
2. **비밀번호 변경** 선택
3. 현재 비밀번호와 새 비밀번호 입력 후 변경

### Q2. 비밀번호를 잊어버렸어요.

현재 비밀번호를 모르는 경우, 관리자에게 문의하여 비밀번호 재설정을 요청하세요.

**관리자 비밀번호 재설정 방법:**
1. 관리자 계정으로 로그인
2. **사용자** 메뉴로 이동
3. 해당 사용자의 **수정** 버튼 클릭
4. 새 비밀번호 입력 후 **저장**
5. 사용자에게 새 비밀번호를 전달 (구두, 메신저 등)

### Q3. 연락처 다운로드 버튼이 보이지 않아요.

편집국 소속 직원 중 부장급 미만은 연락처 다운로드가 제한됩니다. 권한이 필요한 경우 관리자에게 문의하세요.

### Q4. 엑셀 임포트 시 "회사를 찾을 수 없음" 오류가 발생해요.

엑셀 파일의 "소속기업" 컬럼에 입력한 기업명이 시스템에 등록된 기업명과 정확히 일치해야 합니다. 기업 메뉴에서 정확한 기업명을 확인 후 다시 시도하세요.

### Q5. 삭제한 연락처를 복구할 수 있나요?

현재 삭제된 연락처는 복구할 수 없습니다. 삭제 전 신중하게 확인해주세요.

### Q6. 기념일이 캘린더에 표시되지 않아요.

기념일 등록 시 "캘린더 표시" 옵션이 활성화되어 있는지 확인하세요. 기념일 수정 화면에서 해당 옵션을 켜면 캘린더에 표시됩니다.

### Q7. 창립기념일이 자동으로 생성되었는데 수정할 수 있나요?

네, 기념일 메뉴에서 해당 창립기념일을 찾아 수정할 수 있습니다. 단, 기업 정보의 설립일을 변경하면 창립기념일 날짜도 자동으로 업데이트됩니다.

### Q8. 로그인이 안 되고 "잠시 후 다시 시도하세요" 메시지가 나와요.

보안을 위해 3분 내 10회 이상 로그인 실패 시 일시적으로 접근이 제한됩니다. 잠시 기다린 후 다시 시도하거나, 비밀번호가 맞는지 확인해주세요.

### Q9. 부서를 삭제하려는데 삭제가 안 돼요.

부서 삭제는 다음 조건을 모두 충족해야 가능합니다:
- 하위 팀이 없어야 함
- 소속된 사용자가 없어야 함

먼저 해당 부서의 하위 팀을 삭제하고, 소속 사용자를 다른 부서로 이동시킨 후 삭제하세요.

### Q10. 주요 주년(마일스톤) 기념일은 어떻게 표시되나요?

1주년, 5주년, 10주년 등 5년 단위의 주요 기념일은 골드색 뱃지로 강조 표시됩니다. 캘린더에서는 골드색 이벤트 바로, 목록에서는 골드색 주년 뱃지로 구분됩니다.
