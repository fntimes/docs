# FnBook 서비스 개요

서비스 오픈 전 주요 기능 및 권한 체계 확인을 위한 문서.

---

## 1. 서비스 소개

FnBook은 기업 연락처 및 기념일을 관리하는 시스템입니다.

| 기능 | 설명 |
|------|------|
| 연락처 관리 | 기업별 담당자 연락처 등록/수정/검색, 엑셀 임포트/다운로드 |
| 기업 관리 | 상장사/비상장사 기업 정보 관리 |
| 기념일 관리 | 창립기념일, 상장기념일 등 캘린더 관리 및 모니터링 |

---

## 2. 메뉴 구성

| 메뉴 | 기능 | 비고 |
|------|------|------|
| 홈 | 대시보드 (통계, 다가오는 기념일) | - |
| 연락처 | 연락처 목록/등록/수정/삭제, 엑셀 다운로드/임포트 | 다운로드 권한 제한 있음 |
| 기업 | 기업 목록/등록/수정/삭제 | 설립일 입력 시 창립기념일 자동 생성 |
| 기념일 | 캘린더 보기, 기념일 모니터링 (7일/30일/분기) | 마일스톤 주년 강조 표시 |
| 사용자 | 사용자 목록/등록/수정/삭제 | 관리자만 수정 가능 |
| 부서 | 부서 목록/등록/수정/삭제 | 관리자 전용 |
| 더보기 | 비밀번호 변경, 로그아웃 | 모바일 전용 |

---

## 3. 사용자 역할

| 역할 | 설명 |
|------|------|
| 관리자 (Admin) | 모든 기능 접근, 사용자/부서 관리 |
| 부서장 (Department Head) | 일반 기능 + 사용자 메뉴 열람 |
| 팀장 (Team Head) | 일반 기능 (연락처, 기업, 기념일) |
| 기자 (Reporter) | 일반 기능 (연락처, 기업, 기념일) |
| 열람자 (Viewer) | 읽기 전용 |

---

## 4. 권한 상세

### 4.1 메뉴 접근 권한

| 메뉴 | Admin | Dept Head | Team Head | Reporter | Viewer |
|------|:-----:|:---------:|:---------:|:--------:|:------:|
| 홈 | O | O | O | O | O |
| 연락처 | O | O | O | O | O (읽기) |
| 기업 | O | O | O | O | O (읽기) |
| 기념일 | O | O | O | O | O (읽기) |
| 사용자 | O (수정) | O (열람) | X | X | X |
| 부서 | O | X | X | X | X |

### 4.2 연락처 다운로드 권한

편집국 소속 직원의 연락처 다운로드를 제한합니다.

| 조건 | 다운로드 |
|------|:--------:|
| 관리자 | O |
| 편집국 외 부서 | O |
| 편집국 + 부장급 이상 | O |
| 편집국 + 부장급 미만 | **X** |

**편집국 부서**: 금융부, 증권부, 산업부, 유통부, 부동산부

**부장급 이상**: 부장, 국장, 이사, 전무, 사장

### 4.3 데이터 수정 권한

| 대상 | 생성 | 수정 | 삭제 |
|------|:----:|:----:|:----:|
| 연락처 | 모든 사용자 | 모든 사용자 | 모든 사용자 |
| 기업 | 모든 사용자 | 모든 사용자 | 모든 사용자 |
| 기념일 | 모든 사용자 | 모든 사용자 | 모든 사용자 |
| 사용자 | Admin | Admin | Admin |
| 부서 | Admin | Admin | Admin |

> **참고**: Viewer 역할은 모든 데이터에 대해 읽기만 가능합니다.

## 5. 주요 기능 상세

### 5.1 연락처 엑셀 임포트

- 엑셀 양식에 맞춰 연락처 대량 등록
- 업로드 전 미리보기에서 유효성 검증 (오류 항목 표시)
- 필수 항목: 이름, 소속기업

### 5.2 기념일 모니터링

| 구분 | 표시 방식 | 강조 |
|------|----------|------|
| 7일 이내 | 카드 형태 상세 정보 | 빨간색 |
| 30일 이내 | 테이블/리스트 | 주황색 |
| 이번 분기 | 칩 형태 간략 표시 | 파란색 |

### 5.3 마일스톤 주년

5년 단위 주요 기념일(1, 5, 10, 15, 20주년 등)은 골드색 뱃지로 강조 표시됩니다.

### 5.4 로그인 보안

- 3분 내 10회 로그인 실패 시 일시적 접근 제한
